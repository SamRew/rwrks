Dynamic Documents with Knitr

April 9, 3121 Snedecor

Karsten Maurer, karstenm@iastate.edu.
Carson Sievert, sievert@iastate.edu.
Eric Hare, erichare@iastate.edu.

The Dynamic Documents with Knitr workshop will teach you how create reproducible documents containing your code, R output, and text. The course will show you how to create dynamic LaTeX documents and web pages from R which integrate your code, graphs, and data to create truly reproducible research and provide context for your data analysis. This course will also be useful for instructors who wish to create multiple test forms easily.

The course will be a mix of instruction and interactive activities. It will be held in a computer lab but you are encouraged to bring your own laptops, with software already loaded. A list of software will be available at this site several days prior to the workshop.

Learning outcomes

After the end of the this one course, we expect you to be able to do the following:

  • Create LaTeX documents within R
  • Format graphics and R code directly within your latex document
  • Use R packages like xtable and Hmisc to print function output to LaTeX documents
  • Create webpages with R and markdown
  • Understand the advantages of reproducible research tools like knitr to science

Getting Ready on your own machine

You can bring your own laptop and work on it, if you like to. In order to keep up with everybody else, we're asking that you start installing the following pieces of software ahead of coming to the workshop:

  • Install the newest R version (3.0.1 or above) (from www.r-project.org)
  • Install RStudio (from www.rstudio.org)
  • Within R, use "install.packages('knitr')" to install the knitr package used in this workshop
